4. **Hold state pages for all other states** until at least 2 city pages per state are live — Google's quality signals favour state hubs with populated sub-pages.

## 4. Competitor Gap Analysis

### Companies to Monitor

| Competitor | Primary Angle | Alexa / DA Est. | Location Page Pattern to Watch |

| **GoodLeap** | Financing platform, installer network | DA ~52 | `/solar-financing/[state]/` |
| **Tesla Energy** | Powerwall + Solar Roof brand halo | DA ~85 | `/energy/solarpanels/` (no city pages — major gap) |
| **EnergySage** | Marketplace / review aggregator | DA ~62 | `/local-data/[state]/[city]/` |

### How to Run the Gap Analysis

#### Ahrefs (recommended)

1. Open **Site Explorer** → enter `sunrun.com` → **Top Pages** → filter by keyword containing `solar` + `city name`.
2. Export the top 200 location pages ranked in positions 1–20 for keywords containing `no credit`, `financing`, or `bad credit`.
3. Cross-reference against the SR Energy URL inventory (output of `npx tsx scripts/audit-links.ts`).
4. Any URL pattern where a competitor ranks in positions 3–15 for a `[city]` keyword but has **no "no credit check" variant** is an immediate SR Energy opportunity — their page does not address the no-FICO angle.
5. Repeat for all 5 competitors. Track in a shared spreadsheet with columns: Competitor · URL · Keyword · Position · SR Energy equivalent URL · Gap type.

#### Google Search Console (for SR Energy owned data)

1. **Performance** → filter **Queries** containing `no credit` or `no fico` or `VPP`.
2. Sort by **Impressions DESC**. Any query with >100 impressions and CTR <3% is a title/meta description optimisation target — the page is being seen but not clicked.
3. Filter **Pages** to `/locations/texas/*/texas-vpp/` — these pages should generate the highest impression growth in the first 90 days post-launch. If any VPP city page has fewer than 500 impressions after 60 days, submit it manually for re-crawl via the URL Inspection tool.

#### Manual Keyword Gap Checks (free)

Search the following exact queries in an incognito Chrome window and record who ranks in positions 1–5:

```
"solar panels Houston no credit check"
"no fico solar Los Angeles"
"Texas VPP program free battery"
"solar panels New York bad credit"
"free solar battery Texas"
```

If no competitor has a dedicated page for any of these, SR Energy's page is likely to rank within 60–90 days of indexing with correct on-page optimisation.
